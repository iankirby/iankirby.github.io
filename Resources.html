LaTeX

  • github.com/linggrads/latex - a GitHub repository I developed with Hayley Ross for the purposes of introducing LaTeX to members of the Harvard Linguistics department who had not otherwise been introduced formally to typesetting with LaTeX. The repository contains a variety of different types of linguistic documents formatted in LaTeX, showcasing a variety of types of packages.
  • fixBib.pl - a Perl script I wrote which fixes forced lower-casing to non-initial characters in .bib file strings.
  • https://detexify.kirelabs.org/classify.html - draw a symbol you're looking to include in your LaTeX document, it provides you the command, package that you need to include such a symbol.

Old and Middle English

  • Old, Middle English Keyboard with LaTeX Output, a web application I wrote (with ChatGPT's assistance) which allows you to quickly type certain somewhat uncommon characters which are necessary for Old, Middle English (e.g. æ, þ). Provides a keyboard and a simple sequence of buttons that will simultaneously produce text characters and LaTeX code of the string you're typing.
